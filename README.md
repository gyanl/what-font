# Chroma Gen

ChromaGen is a simple random color generator.

### But why?

I made it to learn JavaScript. I suppose you can use it if you want to generate a random color. 

### Features

* random color generation
* one-click hex code copying
* responsive design

### Built with

* love
* html
* css
* javascript
